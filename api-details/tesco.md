## 🔧 Available Endpoints

### 🔎 Product Search (UK)

- **`GET /product-search-by-keyword`**
  Search Tesco UK products using keywords like `"milk"`, `"bread"`, `"snacks"`, `"electronics"`.

---

### 📦 Product Details

- **`GET /product-details`** (UK)
  Retrieve detailed product data using **product ID or URL**.

- **`GET /ireland-product-details`** (Ireland)
  Fetch detailed Tesco Ireland product info via **ID or URL**.

---

### 🗂️ Category Browsing (Paginated)

- **`GET /category-page`** (UK)
  Access paginated product listings using **category ID**.

- **`GET /ireland-category-page`** (Ireland)
  Browse Tesco Ireland categories with full pagination support.

---

### 🧭 Category Management

- **`GET /get-category-id-for-uk`**
  Retrieve full category taxonomy for Tesco UK.

- **`GET /get-category-ids-for-ireland`**
  Get structured category hierarchy for Tesco Ireland.
